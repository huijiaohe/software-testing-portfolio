# SauceDemo 功能测试项目

## 项目简介

本项目基于 SauceDemo 电商网站，完成完整的功能测试实践，主要覆盖用户登录、商品浏览、购物车、下单流程等核心业务场景。

通过本项目，验证系统在正常及异常情况下的功能表现，确保核心流程稳定可靠。

---

##  测试目标

* 验证用户登录功能是否正常
* 验证商品列表展示是否正确
* 验证购物车功能（添加/删除商品）
* 验证下单流程是否完整可用
* 验证异常输入及边界情况处理

---

## 测试环境

* 测试网址：https://www.saucedemo.com/
* 浏览器：Chrome
* 操作系统：Windows

---

## 测试内容

### 1. 登录模块

* 正确账号登录
* 错误密码登录
* 空用户名/密码校验

### 2. 商品模块

* 商品列表加载
* 商品详情查看

### 3. 购物车模块

* 添加商品到购物车
* 从购物车删除商品

### 4. 下单模块

* 填写用户信息
* 完成订单流程

---

## 测试方法

* 功能测试（Functional Testing）
* 等价类划分
* 边界值分析
* 异常场景测试
