# software-testing-portfolio
# 软件测试项目合集

## 项目简介

本仓库包含本人在软件测试方向的实践项目，涵盖功能测试、接口测试、UI 自动化测试以及性能测试，旨在系统提升测试能力，并积累实际项目经验。

---

## 技能概览

* 功能测试（Functional Testing）
* 接口测试（API Testing）
* UI 自动化测试（ pytest）
* 性能测试（JMeter）
* 测试用例设计（等价类、边界值）
* 测试报告编写与缺陷分析

---

## 项目列表

### 1️ SauceDemo 功能测试

目录：`01-saucedemo-test`

**项目内容：**

* 覆盖登录、商品浏览、购物车、下单流程
* 编写测试用例（CSV）
* 输出测试报告与缺陷报告

---

### DummyJSON 接口测试

目录：`02-dummyjson-api-test`

**项目内容：**

* 接口功能测试（增删改查）
* 参数校验与异常场景测试
* 使用 Postman 进行接口验证

---

### SauceDemo UI 自动化测试 

 目录：`03-ui-automation-saucedemo`

**项目内容：**

* 使用 pytest 实现 UI 自动化测试
* 支持批量执行测试用例

---

### 4️ OpenCart 性能测试

📁 目录：`04-JMETER-opencart`

**项目内容：**

* 使用 JMeter 进行性能测试
* 设计不同并发场景（低/中/高负载）
* 分析响应时间、TPS、错误率等指标
